# MarketingSoHigh SEO

The ultimate WordPress SEO plugin by Techno Believe Solutions Ltd.

Full RankMath/Yoast replacement that works standalone (free) or connected to your MarketingSoHigh dashboard for AI-powered marketing.

## Features

- **Meta Tags & Open Graph** — SEO titles, descriptions, OG tags, Twitter cards
- **Schema / JSON-LD** — 30+ schema types with visual builder
- **XML Sitemaps** — Auto-generated with image, video, news, local support
- **Redirections** — 301/302/307/410/451 with regex, auto-redirect on slug change
- **404 Monitor** — Track and fix broken URLs
- **Image SEO** — Auto ALT/Title attributes from templates
- **Breadcrumbs** — PHP function, shortcode, Gutenberg block + schema
- **Internal Linking** — Link counter with AI suggestions (connected mode)
- **Local SEO** — Business info, opening hours, KML sitemap
- **WooCommerce SEO** — Product schema, GTIN/MPN, brand taxonomy
- **IndexNow** — Instant submission to Bing, Yandex, Naver, Seznam
- **Import/Migration** — One-click from Yoast, RankMath, AIOSEO, SEOPress
- **MSH Connector** — AI-powered content publishing, keyword research, site audits

## Requirements

- WordPress 6.0+
- PHP 8.0+
